# evaluation-service.md

## 1. Purpose

Evaluation Service is the quality and validation layer of the platform.

---

## 2. Example checks

- graph completeness
- graph runtime consistency
- scoring consistency
- report validity
- regression checks

---

## 3. Why it matters

Even in MVP, evaluation is useful as an explicit architectural place for:
- QA
- graph validation
- safety checks
- future automated testing
